# Landing

Одностраничный лендинг для ведущей + защищенная админка для загрузки фото/видео и управления отзывами.

## Что есть

- Лендинг с секциями и формой заявок.
- Админка `/admin/login` с парольным входом.
- Защита админки: `HttpOnly` сессия, CSRF-токен, лимит неудачных попыток входа.
- Серверные API для загрузки медиа, управления отзывами, экспорта и очистки данных.
- Публичный лендинг автоматически показывает видео/фото и отзывы, загруженные в админке.

## Быстрый старт

```bash
pnpm install
copy .env.example .env
pnpm admin:hash -- yourStrongPassword123
```

1. Вставьте результат `pnpm admin:hash` в `ADMIN_PASSWORD_HASH` в `.env`.
2. Установите `SESSION_SECRET` (минимум 32 символа, лучше случайная строка).
3. Запустите проект:

```bash
pnpm dev
```

Админка:

- Вход: `http://localhost:4321/admin/login`
- Панель: `http://localhost:4321/admin`

## Хранение данных админки

- Метаданные: `storage/admin/content.json`
- Загруженные файлы: `public/uploads/`

## Команды

```bash
pnpm dev
pnpm build
pnpm preview
pnpm admin:hash -- <password>
```

## Форма заявки

Форма отправляет данные на FormSubmit. При первом запросе сервис попросит подтвердить email.  
Маршрут успешной отправки: `/thanks`.
